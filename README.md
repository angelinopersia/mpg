# mpg

This is the technical test for Mon Petit Gazon.

# Installation

- Clone the repo

```bash
git clone https://github.com/angelinopersia/mpg.git
```

- Install the packages

```bash
cd mpg
yarn
```

#### Run on Android

- Start metro

```bash
npx react-native start
```

- Run the project

```bash
npx react-native run-android
```
